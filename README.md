# PackageKit-Qt

Qt5 bindings for PackageKit


## Install

```
cd PackageKit-Qt/
mkdir -p build
cd build
cmake ..
make
sudo make install
```


## pkg-config

```
pkg-config --cflags --libs packagekit-qt5 
```
